# Django Project

This repository contains a full‑stack web application built with the **Django** framework (Python) for the backend and a TypeScript/React‑styled frontend in `src/pages`. The project implements user login, registration, and task management UI pages. It was created as a personal learning project and demonstrates basic full‑stack web development with Django and a modern frontend.  

---

## 🧱 Features

✔ User Authentication (Login & Register)  
✔ Task management pages (Add Task, Edit Task, Task List)  
✔ React/TypeScript frontend with UI forms  
✔ Uses Django as the web server (backend API and templates)

---

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Requirements

Make sure you have installed:

- Python 3.8+  
- pip (Python package manager)  
- `virtualenv` (optional but recommended)  
- Node.js & npm/Yarn (if the frontend requires compiling)
